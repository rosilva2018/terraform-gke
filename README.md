# IaC GKE

Este projeto entrega um cluster de Kubernetes GKE, construído com Terraform, e uma aplicação simples com uma interface de usuário (UI) para manipulação da API do Kubernetes.

# Uso

### App
cd /app.
node index.js

### Infra
terraform init
terraform plan
terraform apply

# Dependências
* Node.js    - v18.9.0
* npm 8.19.1 - 8.19.1
* Terraform  - 1.5.7
